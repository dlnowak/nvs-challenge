NVS Digital Challenge files

This site hosts the files from the NVS Digital Challenge on a real Web server (as opposed to the GitHub repository, which imposes various restrictions and does not provide correct MIME types). Feel free to use these directly in your project—“hotlink,” make AJAX requests, etc.

For more information about the NVS Digital Challenge, please visit the announcement page.
